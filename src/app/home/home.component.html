Differential Analysis (AlphaPeptStats Generic) A Differential analysis workflow that requires a single input data set with a minimum of two conditions. The workflow utilize the AlphaPeptStats Python package with Generic loader.
Differential Analysis (QFeatures) A Differential analysis workflow that requires a single input data set with a minimum of two conditions. The workflow utilize the QFeatures R package.
Imputation (QFeatures) Data imputation to remove NA or blank data points from the datatable . The workflow utilize the QFeatures R package.
Normalization (QFeatures) Data normalization to allow differential analysis between data of different conditions. The workflow utilize the QFeatures R package.
Correlation Matrix (R) Create correlation matrix plot using the corrplot package from R.
